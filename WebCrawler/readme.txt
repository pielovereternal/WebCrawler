We provide some skeleton code to get you started.  It is incomplete
and not of the highest quality.  Your assignment submission, naturally,
will be complete and of the highest quality.

The following files are included:

Crawler.java               abstract class from which WebCrawler is derived
WebCrawler.java            your crawler goes here. There is some sample code here as well
Index.java                 a class with serialization code
WebIndex.java              your index goes here
WebQueryEngine.java        your query engine goes here
WebQueryEngineApp.java     part of the web interface
index.html                 part of the web interface
tsoogle.gif                part of the web interface
readme.txt                 this file
tagsoup-1.2.1.jar          a jar for HTML parsering, used by Crawler.java
